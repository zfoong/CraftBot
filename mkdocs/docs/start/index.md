## Prerequisites

- Python **3.10+** — check with `python --version`
- `git` to clone the repo
- An API key from **OpenAI**, **Google Gemini**, **Anthropic**, **BytePlus**, or a local **Ollama** server
- *(optional)* Node.js 18+ for browser mode

## Recommended reading order

1. [Install](install.md)
2. [Quickstart](quickstart.md) — runs through onboarding automatically
3. [Your first task](hello-task.md)
4. Pick an interface from [Interfaces](../interfaces/index.md)
5. (Optional) Set up [service mode](service-mode.md) for always-on operation
