    Getting started¶

    Install CraftBot, run through onboarding, send your first task, and optionally set it up as a background service. Five pages, roughly 15 minutes end-to-end.

    • Install

      Python 3.10+, python install.py, and your LLM API key. Windows, Linux, and macOS.

    • Quickstart

      From zero to a running agent in 5 minutes. Browser, TUI, or CLI.

    • Onboarding

      First-launch wizard: agent name, API keys, skills, MCP servers.

    • Your first task

      Send a message, watch CraftBot plan and execute.

    • Service mode

      Run CraftBot as an always-on background service with auto-start.

    Prerequisites¶

    • Python 3.10+ — check with python --version
    • git to clone the repo
    • An API key from OpenAI, Google Gemini, Anthropic, BytePlus, or a local Ollama server
    • (optional) Node.js 18+ for browser mode

    Recommended reading order¶

    1. Install
    2. Quickstart — runs through onboarding automatically
    3. Your first task
    4. Pick an interface from Interfaces
    5. (Optional) Set up service mode for always-on operation
